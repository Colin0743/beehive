## 📁 项目结构

```
beehive-platform/
├── src/
│   ├── app/                      # Next.js 页面
│   │   ├── auth/                # 认证页面
│   │   │   ├── login/          # 登录
│   │   │   └── register/       # 注册
│   │   ├── projects/           # 项目相关
│   │   │   ├── new/           # 创建项目
│   │   │   ├── edit/[id]/     # 编辑项目
│   │   │   └── [id]/          # 项目详情
│   │   ├── profile/           # 个人中心
│   │   ├── layout.tsx         # 根布局
│   │   ├── page.tsx           # 首页
│   │   └── globals.css        # 全局样式
│   ├── components/            # React 组件
│   │   ├── HeaderSimple.tsx   # 导航栏
│   │   └── LayoutSimple.tsx   # 页面布局
│   └── contexts/              # Context
│       └── AuthContext.tsx    # 认证上下文
├── public/                    # 静态资源
│   └── default-avatar.svg     # 默认头像
└── 配置文件
```

## 📊 数据存储

使用 localStorage 存储以下数据：

| 键名 | 说明 |
|------|------|
| `user` | 当前登录用户 |
| `registeredUsers` | 所有注册用户 |
| `projects` | 所有项目 |
| `userProjects_{userId}` | 用户创建的项目 |
| `followedProjects_{userId}` | 用户关注的项目 |
| `participatedProjects_{userId}` | 用户参与的项目 |
