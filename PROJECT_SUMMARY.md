## 项目概述

"蜂巢"是一个AI视频协作平台，旨在连接AI视频创作者，促进创意与算力的汇聚。

## 技术实现

### 技术栈
- **前端框架**: Next.js 16 (App Router)
- **编程语言**: TypeScript
- **UI库**: Semantic UI React + Tailwind CSS
- **状态管理**: React Context API
- **数据存储**: localStorage (MVP阶段)

### 项目结构
```
beehive-platform/
├── src/
│   ├── app/                 # 页面路由
│   ├── components/          # 可复用组件
│   └── contexts/            # Context状态管理
├── public/                  # 静态资源
└── 配置文件
```

## 本地部署

### 访问地址
- **开发服务器**: http://localhost:3001
- **网络访问**: http://192.168.2.5:3001

### 启动命令
```bash
cd beehive-platform
npm install
npm run dev
```

## 数据存储方案

使用 localStorage 存储以下数据：
- `user`: 当前登录用户
- `registeredUsers`: 注册用户列表
- `projects`: 所有项目
- `userProjects_{userId}`: 用户创建的项目
- `followedProjects_{userId}`: 用户关注的项目
- `participatedProjects_{userId}`: 用户参与的项目
