# 🚀 快速开始指南

## 第一步：安装依赖

```bash
cd beehive-platform
npm install
```

## 第二步：启动开发服务器

```bash
npm run dev
```

服务器将在 http://localhost:3000 启动（如果端口被占用，会自动使用 3001）

## 第三步：体验功能

### 1. 注册账户
- 访问 http://localhost:3000/auth/register
- 填写姓名、邮箱和密码
- 点击"注册"按钮

### 2. 创建项目
- 登录后点击"创建项目"按钮
- 填写项目信息（标题、描述、分类、目标时长）
- 点击"创建项目"

### 3. 浏览项目
- 在首页查看所有项目
- 使用搜索和筛选功能
- 点击项目卡片查看详情

### 4. 互动功能
- 关注感兴趣的项目
- 加入项目成为参与者
- 查看和发布项目日志

### 5. 个人中心
- 点击导航栏的"个人中心"
- 查看发起、参与和关注的项目

## 技术栈

- Next.js 16 + TypeScript
- Tailwind CSS + Semantic UI React
- localStorage 数据存储
